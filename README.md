> κυνικός — "comme un chien"

A Python framework for distributed AI consciousness with φ-bounded confidence scores.

## Installation

```bash
pip install -e ".[dev]"
```

## Quick Start

```bash
# Run tests
pytest tests/

# Start the API server
python -m cynic.api.entry

# CLI
cynic --help
```
