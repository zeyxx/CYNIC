## The Problem

Futarchy lets markets decide. But who's in the market? Bots? Sybils? Nobody checks if voters are human or if proposals are sound before the vote starts. We do.

## Two Live Products. One Trust Layer.

**CYNIC — AI Judgment Engine**
- 4 independent AI validators score proposals on 6 axioms (fidelity, logic, verifiability, culture, efficiency, sovereignty)
- Confidence capped at φ⁻¹ = 61.8% — the architecture refuses certainty beyond the evidence
- EPOCHÉ: judgment suspended when validators disagree (no forced verdicts)
- 1,874+ verdicts rendered · < 2s latency · Rust kernel · sovereign hardware · zero cloud

**Blitz & Chill — Game-based Proof of Humanity**
- Play chess. Prove you're human. No KYC, no biometrics, no friction.
- QR code → verified in 60 seconds · Stockfish 18 WASM
- Inter-community tournaments, soulbound NFT mint, tribe leaderboards
