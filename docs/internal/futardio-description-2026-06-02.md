## The Problem

Every online system asks: *is this a bot?* The answer is always friction — CAPTCHAs, biometrics, KYC. And in governance markets, nobody checks if proposals are sound before trading starts.

## The Solution: TALARIA

Two live products. One trust layer.

**Blitz & Chill — Chess for everyone. Human by default.**

The best anti-captcha is one people actually want to use.

- Real-time multiplayer chess for beginners and pros — live now
- Beat the bot → you're verified human. No KYC, no biometrics, no friction.
- Inter-community tournaments: "$TOKEN_A DESTROYED $TOKEN_B 7-3" — self-propagating content, zero ad spend
- Mobile-first, Solana-native, Stockfish 18 WASM engine

**CYNIC — AI Judgment Engine**

- Heuristic + up to 3 independent LLM validators score proposals on 6 axioms
- Confidence capped at φ⁻¹ = 61.8% — the architecture refuses certainty beyond the evidence
- EPOCHÉ: judgment suspended when validators disagree (no forced verdicts)
- 3,711 verdicts rendered · Rust kernel · sovereign hardware · zero cloud dependency
