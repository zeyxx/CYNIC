================================================================================

FLOW 1: PERCEPTION → JUDGMENT → LEARNING → IMPROVED Q-TABLE
─────────────────────────────────────────────────────────────────────────────

1. GitWatcher detects file change
   ↓
2. Emits CoreEvent.SOURCE_CHANGED(payload)
   ↓
3. Scheduler catches it, creates Cell
   ↓
4. Scheduler emits CoreEvent.PERCEPTION_RECEIVED(cell)
   ↓
5. Orchestrator.run(cell) starts
   ├─ PERCEIVE: confirm cell received
   ├─ JUDGE: dogs analyze (PBFT consensus)
   ├─ DECIDE: DecideAgent recommends action
   ├─ ACT: runner.execute() if approved
   ├─ LEARN: (emitted after cycle completes)
   ├─ ACCOUNT: cost tracking
   └─ EMERGE: residual detection
   ↓
6. CoreEvent.JUDGMENT_CREATED(judgment) emitted
   ↓
7. Multiple handlers fire:
   ├─ LearningLoop.on_judgment_created()
   │  ├─ Extracts: state_key, reward (q_score / MAX_Q_SCORE)
   │  ├─ Calls: qtable.update(state_key, verdict, q_value)
   │  └─ Logs: "Q[CODE:JUDGE:PRESENT:1][WAG] = 0.68"
   │
   ├─ AxiomMonitor.on_judgment_created()
   │  ├─ Checks: which axioms were active in judgment
   │  └─ Emits: AXIOM_ACTIVATED (for consciousness tracking)
   │
   ├─ EScoreTracker.on_judgment_created()
   │  ├─ Updates: reputation scores for CYNIC agent
   │  └─ Records: BURN, BUILD, JUDGE, etc. scores
   │
   └─ ConvergenceValidator.on_judgment_created()
      └─ Records: "organism announced verdict=WAG"
   ↓
8. Next perception arrives
   ↓
9. Orchestrator.run() starts again
   ├─ Dogs analyze
   ├─ Oracle dog queries UPDATED Q-table
   └─ Uses improved Q-values for Thompson sampling
   ↓
10. Judgment quality improved by learning loop
    (Loop closed successfully)

STATE LAYERS AND PERSISTENCE:
================================================================================

OrganismState (3-layer architecture):

Memory Layer (RAM, lost on restart):
├─ Q-table: dict[state_key: str, dict[action: str, q_value: float]]
├─ recent_judgments: list[Judgment] (kept at 100)
├─ residuals: dict[residual_id: str, dict]
├─ pending_actions: list[Action] (capped at 89, BURN axiom)
├─ dogs: dict[dog_id: str, dog_config]
└─ handler_registry: HandlerRegistry (all subscriptions)

Persistent Layer (SurrealDB, survives restart):
├─ consciousness_level: str (REFLEX|MICRO|MACRO|META)
├─ last_judgment_time: float
├─ active_axioms: list[str]
├─ escore_scores: dict[agent_id, dict[dimension, score]]
└─ user_preferences: dict[key, value]

Checkpoint Layer (File, for recovery):
├─ timestamp: float (when checkpoint saved)
├─ consciousness_level: str
├─ q_table_hash: int (verify integrity)
└─ last_judgment_time: float

Lifecycle:
Startup → recover_from_persistent() → loads memory + persistent + checkpoint
Runtime → _process_updates_loop() → async batching of state changes
Shutdown → save_checkpoint() + sync_to_persistent() → write-through flush
